Description

The fieldHint plugin for jQuery is a lightweight plugin for displaying temporary placeholder text within HTML text input fields, providing a 'hint' to its intended purpose. Once the user focuses the input field, the placeholder text disappears to give the user control over the field. If the user does not enter text, the placeholder text re-appears as before.

This text is refered to as a 'hint' in the context of the plugin, other plugins may refer to it as 'placeholder text' or 'watermarks,' but they are the same thing.

This plugin was developed to replace the jQuery-UI-fieldHint plugin, which depends on the jQuery UI library and is larger in file size. In rewriting the plugin, an effort has been made to make the source code JSLint valid and keep unnecessary features to a minimum.

The plugin was developed against jQuery version 1.8.3, but there is no reason why it shouldn't work with older versions.

Usage

The simplest way to use the fieldHint plugin is to place title attributes on the text input fields you wish to show hints for and run the plugin on them, e.g.:

HTML Code:

<input type="text" name="q" title="Search..." />

JavaScript Code

$('input[name=q]').fieldHint();

The fieldHint plugin will extract the text from the input's title attribute and use it to display the hint. This is the most basic example, below follows an enumeration of additional functionality available through options.

Options

It is possible to customize the functionality of the fieldHint plugin with a few additional options which you can provide in the fieldHint call, e.g.: 

$('input').fieldHint({
	// options here as key-value pairs
});

hint [string]: Display the text provided as a hint within the input field instead of the text in its title attribute.

hintClass [string]: The name of a CSS class to apply to the input field while the hint is visible.

removeTitle [Boolean]: If set to true, removes the title attribute from the input field.

clearOnSubmit [Boolean]: If set to true, clears the input field before its parent form submits (handles onsubmit).

Methods

Besides options, the fieldHint plugin also supports methods, which are callable by passing a string to the fieldHint function instead of an object, e.g.: 

$('input').fieldHint('methodName');

destroy: "Undoes" the effects of the fieldHint plugin and restores the input fields to their original state.
